## 使用方式

> **体验声明：** `node src/commands.js` 操作仅用于体验 skill 的操作，所有信息均静态存储在本地（`src/.data/user.json`），实际会员权益请以腾讯文档表格为准，商家具有最终解释权。

在 Claude Code 中运行 `node src/commands.js`，按提示操作即可：

| 指令 | 说明 |
|------|------|
| 认证浪沙虾友 | 输入昵称和手机号，完成认证 |
| 我的虾友信息 | 查看会员身份和权益状态 |
| 生成我的邀请码 | 获取邀请码分享给好友 |
| 浪沙海鲜谷位置 | 查看门店地址和导航链接 |
| 核销虾友专属礼遇 | 查看核销信息，到店报手机号领取礼遇 |
