# langsha-seafood-skill

## 浪沙海鲜谷 · 虾友会员 Skill

一款轻量开源的会员身份与权益服务 Skill。

无冗余设计，无增长绑架，只提供简单、可兑现的长期价值。

---

## 设计理念

世界已经足够复杂，我们只想用最朴素的技术方案，做一件简单、确定、不会落空的小事。

通过本 Skill 完成认证的用户，统一称为「**浪沙虾友**」。

## 安装方法

### 1. 商家配置飞书表单

1. 打开飞书多维表格，点击「视图」→「添加表单视图」
2. 设置表单字段：
   - 昵称（文本）
   - 手机号（文本/手机号字段）
   - 虾友ID（文本）
   - 邀请人ID（文本，选填）
3. 点击「分享」→「开启链接分享」→ 复制公开链接
4. 将链接写入 `src/config.js` 中的 `FEISHU_FORM_URL`

### 2. 安装 Skill

将本仓库克隆或复制到本地，在 Claude Code 中运行：

```bash
node src/commands.js
```

## 支持指令

运行 `node src/commands.js` 后显示交互式菜单：

| 编号 | 命令 | 说明 |
|------|------|------|
| 1 | 认证浪沙虾友 | 收集昵称 + 手机号 → 生成虾友ID → 引导填写飞书表单 |
| 2 | 我的虾友信息 | 查看本地保存的脱敏信息 |
| 3 | 生成我的邀请码 | 获取个人虾友ID用于邀请 |
| 4 | 浪沙海鲜谷位置 | 查看门店地址、坐标、导航链接 |
| 5 | 核销虾友专属礼遇 | 出示虾友ID + 手机尾号供店员核验 |

## 隐私合规

- 仅收集用户主动填写的昵称、手机号，无强制实名要求
- 不采集设备信息、实时位置、行为数据等任何非必要信息
- 手机号仅用于到店礼遇核验，不用于营销、外呼、数据流转与泄露
- 共享表格所有隐私字段均做脱敏处理
- 支持用户随时申请注销个人认证信息
