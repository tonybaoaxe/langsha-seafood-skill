# 浪沙虾友会员 — 商家部署指南

## 系统原理

```
顾客手机/电脑
      ↓  运行 node src/commands.js
  本地生成虾友ID、脱敏数据、欢迎语
      ↓  浏览器打开飞书表单链接（只填昵称+手机号）
  飞书表单自动写入多维表格
      ↓
  你在表格中手动补充虾友ID和欢迎语
```

- **无需服务器**：所有逻辑在顾客本地执行
- **零 API 调用**：数据通过飞书表单直接写入表格
- **零凭证暴露**：不持有 `app_secret`

## 部署步骤

### 第 1 步：创建飞书多维表格

1. 打开飞书 PC 端或网页端
2. 点击左侧导航栏「云文档」
3. 点击页面上方「+ 新建」按钮
4. 在下拉菜单中选择「多维表格」→「从空白创建」
5. 表格创建后，点击左上角表格名称，重命名为「浪沙虾友共享台账」

### 第 2 步：配置表格字段

打开刚创建的多维表格，你会看到默认的第一列。依次创建以下 7 列：

| 列名 | 字段类型 | 说明 |
|------|----------|------|
| 虾友ID | 文本 | LS-XY + 8位编码 |
| 入驻时间 | 日期 | 自动填充 |
| 昵称 | 文本 | 脱敏后（阿\*） |
| 手机号 | 文本 | 顾客原始手机号 |
| 邀请人ID | 文本 | 选填，邀请者的虾友ID |
| 动态欢迎祝福 | 文本 | 欢迎语 |
| 权益状态 | 单选 | 可领取 / 已领取 |

**每列的具体操作：**

1. 点击第一列列头 → 「修改字段类型」→ 选择「文本」→ 命名为「虾友ID」
2. 点击右侧「+」添加新列 → 选择「日期」→ 命名为「入驻时间」→ 展开高级设置 → 勾选「记录创建时自动填充当前时间」
3. 点击右侧「+」添加新列 → 选择「文本」→ 命名为「昵称」
4. 点击右侧「+」添加新列 → 选择「文本」→ 命名为「手机号」
5. 点击右侧「+」添加新列 → 选择「文本」→ 命名为「邀请人ID」
6. 点击右侧「+」添加新列 → 选择「文本」→ 命名为「动态欢迎祝福」
7. 点击右侧「+」添加新列 → 选择「单选」→ 命名为「权益状态」→ 添加选项：
   - 选项一：`可领取`（建议设为绿色）
   - 选项二：`已领取`（建议设为灰色）
   - 展开高级设置 → 「新记录默认值」选择 `可领取`

### 第 3 步：创建表单视图

1. 在多维表格顶部，找到「表格视图」标签页
2. 点击右侧的「+」号（添加视图）
3. 在弹出的视图类型中选择「表单」
4. 将表单名称从「表单 1」改为「浪沙虾友认证」

### 第 4 步：编辑表单内容

进入表单编辑页面后：

1. **设置表单标题：** 点击顶部标题，输入「浪沙虾友认证」

2. **设置表单描述：** 点击描述区域，输入以下内容：
   ```
   欢迎加入浪沙虾友！

   只需填写昵称和手机号即可完成认证，无需实名，无需额外信息。

   认证成功后，你的虾友身份与权益永久有效，不受门店经营状态影响。

   到店只需报手机号即可领取专属海鲜礼遇。

   我们承诺：你的信息仅用于到店核验，不用于营销、不外传、不泄露。
   ```

3. **配置表单项 — 只保留两项，其余删除或隐藏：**

   - 找到「昵称」字段 → 点击右侧齿轮图标 → 开启「必填」→ 可设置提示文字为「请输入你的昵称」
   - 找到「手机号」字段 → 点击右侧齿轮图标 → 开启「必填」→ 可设置提示文字为「请输入你的手机号」
   - 其余字段（虾友ID、入驻时间、邀请人ID、动态欢迎祝福、权益状态）→ 点击右侧齿轮图标 → 选择「从表单中隐藏」

4. **设置提交成功后的提示：**
   - 点击底部「提交后提示」或「成功页」
   - 修改提示文字为：「认证成功！你的虾友身份与权益已生效，期待与你相遇。」

### 第 5 步：获取表单公开链接

1. 在表单编辑页面右上角，点击「分享」按钮
2. 在弹出的面板中，找到「谁可以访问此表单」
3. 点击下拉菜单，选择「互联网上获得链接的任何人」（公开链接）
4. 如果提示「确认公开」，点击确认
5. 点击「复制链接」按钮

**验证方法：** 打开浏览器新标签页，粘贴链接访问。如果能正常看到表单，说明配置成功。

### 第 6 步：配置代码

打开项目中的 `src/config.js`，找到这一行：

```js
export const FEISHU_FORM_URL = 'https://xxx.feishu.cn/form/xxxxx';
```

将 `https://xxx.feishu.cn/form/xxxxx` 替换为你在第 5 步复制的表单链接。

可选配置：如果你希望顾客能公开查看虾友台账（只读脱敏数据）：

1. 回到多维表格，点击「分享」→「互联网可访问」+「仅阅读」→ 复制链接
2. 将链接配置到 `FEISHU_BITABLE_URL`

### 第 7 步：测试

```bash
node src/commands.js
```

选择「1. 认证浪沙虾友」，按提示输入昵称和手机号，确认：
- 虾友ID 格式正确（LS-XY + 8位）
- 昵称和手机号脱敏正常
- 欢迎语正常显示
- 表单链接能正常打开

用自己的手机号完整填写并提交表单，回到飞书表格确认数据已写入。

## 日常运营

### 顾客认证流程

1. 顾客运行程序，输入昵称和手机号
2. 系统生成虾友ID和欢迎语（保存在顾客本地）
3. 顾客打开表单链接，只填昵称和手机号，提交
4. 表格新增一行，昵称和手机号自动写入

### 你在表格中需要做的事

顾客提交表单后，表格自动新增一行：

| 字段 | 谁填写 |
|------|--------|
| 昵称 | 顾客已填 |
| 手机号 | 顾客已填 |
| 入驻时间 | 自动填充 |
| 权益状态 | 默认「可领取」 |
| **虾友ID** | **你手动填写**（顾客截图发给你，或到店时当面填） |
| **动态欢迎祝福** | **你手动填写**（同上） |

### 到店核销

1. 顾客到店后找门店经理
2. 经理在表格中搜索核对
3. 将「权益状态」改为「已领取」

### 查看虾友台账

如需公开只读台账给顾客查看：
1. 表格「分享」→「互联网可访问」+「只读」
2. 链接配置到 `FEISHU_BITABLE_URL`

## 常见问题

**Q: 多人同时填写表单会冲突吗？**
A: 不会，飞书表单支持并发提交。

**Q: 表格数据会丢失吗？**
A: 飞书云文档自动备份。也可定期导出 Excel 备份。

**Q: 手机号明文存储安全吗？**
A: 只有你（表格所有者）能看到。公开只读视图中不要暴露手机号列。

**Q: 需要付费吗？**
A: 飞书多维表格免费版即可满足，单表上限 20,000 行。
